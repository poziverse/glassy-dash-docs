## ğŸ› Reporting Bugs

### Before Reporting
- [ ] Check [FAQ](GLASSYDASH/docs/user/FAQ.md) for common questions
- [ ] Search [existing issues](https://github.com/yourusername/glassy-dash/issues)
- [ ] Try the [latest version](https://github.com/yourusername/glassy-dash/releases)

### How to Report
Use our [Bug Report Template](https://github.com/yourusername/glassy-dash/issues/new?template=bug_report.md):

1. **Title**: Clear, descriptive bug title
2. **Description**: What happened, what you expected
3. **Steps to Reproduce**: Step-by-step instructions
4. **Environment**: OS, Node.js version, browser
5. **Screenshots**: If applicable
6. **Logs**: Relevant error messages

### Good Bug Report Example
```
Title: Notes not saving when creating checklist items

Description:
When creating a new note with checklist items, clicking "Save" 
doesn't persist the checklist items to the database.

Steps to Reproduce:
1. Click "Add Note"
2. Select "Checklist" type
3. Add 3 checklist items
4. Click "Save"
5. Close modal
6. Open the note again

Expected: Checklist items should be visible
Actual: Checklist items are missing

Environment:
- OS: Ubuntu 22.04
- Version: v0.67
- Browser: Chrome 120

Logs:
Error: INSERT into checklist_items failed - FOREIGN KEY constraint
```

### When Asking for Help
1. **Be specific** - Include error messages, screenshots, code
2. **Provide context** - What are you trying to do?
3. **Show research** - What have you already tried?
4. **Use code blocks** - Format code and logs properly
5. **One issue per post** - Don't bundle multiple problems

### Good Help Request Example
```
Title: Error when importing Google Takeout JSON

I'm trying to import notes from Google Keep but getting this error:

3. Cleared browser cache

Here's the JSON file:

  "content": "..."
}
```

Any ideas what I'm doing wrong?
